# 银河系模拟 - Galaxy Simulation

使用 WebGL 和 Three.js 创建的逼真银河系模拟，实现了符合天文观测的旋转效果。

## 特性

- 🌌 逼真的螺旋星系结构
- 🔄 符合天文观测的旋转曲线（中心到边缘不同旋转速度）
- ✨ 50,000+ 粒子系统
- 🎨 美丽的视觉效果（泛光、色调映射）
- 🎮 交互式控制面板

## 技术栈

- Three.js - WebGL 渲染引擎
- GLSL 着色器 - GPU 加速计算
- Vite - 开发构建工具

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 物理模型

### 旋转曲线

实现了符合天文观测的旋转速度函数：
- **中心区域**（0-2 kpc）：速度随距离线性增加
- **中间区域**（2-8 kpc）：速度相对稳定（~220 km/s）
- **边缘区域**（8+ kpc）：速度略微下降但仍保持较高速度（暗物质效应）

### 螺旋臂结构

使用对数螺旋线方程生成自然的螺旋臂结构，在螺旋臂上分布更多粒子。

## 控制说明

- **鼠标拖拽**：旋转视角
- **滚轮**：缩放
- **控制面板**：调整旋转速度、粒子数量、视角预设

## 浏览器支持

需要支持 WebGL 2.0 的现代浏览器。
